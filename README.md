## Despliegue del proyecto

Pasos para despliegue:

- Clonar proyecto desde git o copiar archivos (git clone https://...).
- Instalar dependencias Laravel ejecutando desde CMD en la raiz del proyecto el comando "componser install"
- Instalar dependencias de vistas ejecutando desde la raiz del proyecto el comando "npm install"
- Compilar vistas ejecutando desde la raiz del proyecto el comando "npm run prod"
- asegurarse que se cree en la raiz un archivo .env , usar el archivo de .env.example como guia para configurar el archivo.
